# ⚖️ 法律时光机

> 先看结局，再决定要不要开始。

AI 驱动的诉讼决策辅助工具。上传案情信息和证据，AI 模拟推演诉讼过程，给出胜率预测、成本分析和策略建议。

## 功能

- **胜率预测** — SVG 环形仪表盘动画，三级标签（建议起诉 / 需补充证据 / 建议和解）
- **成本收益计算器** — 诉讼费按法院阶梯计算 + 律师费估算，直接给出净收益结论
- **证据补强清单** — 按纠纷类型匹配缺失证据，标注关键 / 重要 / 辅助优先级
- **对方抗辩模拟** — 模拟对方律师策略，附应对方案
- **律师匹配** — 一键匹配专业律师

## 快速开始

```bash
# 安装依赖
npm install

# 启动服务
npm start

# 开发模式（自动重启）
npm run dev
```

访问 `http://localhost:3000`

## API

```
POST /api/analyze
Content-Type: multipart/form-data

参数：
  caseType    - 纠纷类型（民事纠纷/合同纠纷/劳动纠纷/刑事辩护/其他）
  description - 案情描述
  amount      - 标的金额（元）
  files       - 证据文件（最多 5 份）

返回：胜率、成本、证据清单、抗辩策略等完整分析结果
```
